Utilitza aquest command per crear un contenidor. El que esta després de launch és la versió que vols crear, i al final es el nom que vols crear.

lxc launch ubuntu:18.04 myweb

Utilitza aquest command per executar, el bash et permet escriure commandes.

lxc exec myweb bash

Utilitza aquestes comandes per actualitzar i instal·lar el OwnCloud.

Sudo apt-get update
Sudo apt-get install owncloud-files
